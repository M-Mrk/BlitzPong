## 10/24/2025 - Choose target components and came up with basic idea  

I firstly came up with the idea and how it would generally be laid out.

This project now consists of multiple rows of target, where each row has 5 targets and a controller. The targets and controller are connected via 6 pin pogo connectors (see README).

The targets will be shown as focused targets, via their 16 RGB LED ring. The targets get periodically polled by the main controller via I2C. 

The rows will communicate via ESP-NOW (at least currently). The controllers will have an oled screen and rotary encoder for user input.
